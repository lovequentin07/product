# 아파트 매매 실거래가 공개 자료(상세) API 명세

## 기본 정보
- **상세기능**: 아파트 매매 실거래가 공개 자료(상세)
- **Endpoint**: `/getRTMSDataSvcAptTradeDev`
- **설명**: 부동산 거래신고에 관한 법률에 따라 신고된 거래의 실거래 자료를 제공
- **일일 트래픽**: 10,000회

## 요청 변수 (Request Parameters)
| 항목명 | 샘플데이터 | 설명 |
| :--- | :--- | :--- |
| LAWD_CD | 11110 | 지역코드, 각 지역별 코드 행정표준코드관리시스템(www.code.go.kr)의 법정동코드 10자리 중 앞 5자리 |
| DEAL_YMD | 201512 | 계약월, 실거래 자료의 계약년월(6자리) |
| serviceKey | - | 인증키(URL Encode), 공공데이터포털에서 요청하는 인증키 |
| pageNo | 1 | 페이지번호 |
| numOfRows | 10 | 한 페이지 결과 수 |

## 특이사항
- 호출 속도가 너무 빠를 경우 API 차단 위험이 있으므로 적절한 딜레이(Delay)가 필요함.
- 한 번의 호출로 해당 월의 모든 데이터를 가져오기 위해 `numOfRows`를 적절히 조절함.
